HTML5 이미지(Images)

HTML 이미지 구문

HTML에서 이미지는 <img>로 정의한다.
<img> 태그는 종료 태그가 없으며, src 속성(이미지 URL)을 항상 포함 시켜야된다.

대체 속성

alt 속성은 사용자가 이미지를 볼 수 없는경우(연결이 느릴때, src 속성의 오류 또는 화면 판독기를 사용하는 경우) 이미지에 대체 텍스트를 제공한다.
즉, alt 속성 값은 이미지를 설명해야 된다.

브라우저가 이미지를 찾을 수 없으면 alt속성 값이 표시된다.

웹페이지에 alt 속성이 없으면 유효성이 올바르게 검사되지 않는다.

stlye 속성을 사용하여 이미지 너비(width)와 높이(height) 를 지정 할 수 있다.

또는 width및 height속성을 사용할 수 있다.

너비(width)와 높이(height)지정하지 않으면 이미지가로드되는 동안 페이지가 깜빡일 수 있기 때문에, 항상 이미지의 너비(width)와 높이(height)를 지정해야된다.

너비와 높이 속성 또는 스타일 속성

width, height 속성은 HTML에 적용 할 수 있지만, style 속성을 사용하여 이미지의 크기를 변경하지 못하게 하는 것이 좋다.

다른 폴더의 이미지

이미지 폴더를 지정하지 않으면 동일안 폴더내에서 이미지를 검색한다. 하위 폴더에 이미지를 저장하고 src속성으로 폴더 이름을 포함시키면 된다.

이미지 링크

이미지를 링크로 사용하려면 <img>태그 안에 <a>를 넣으면 된다.


See the Pen

IE9 및 이전 버전은 이미지 링크인 경우 이미지 주위에 테두리(border:0;)를 표시하지 않도록 추가 되었다.

HTML <picture> 요소

<picture>요소는 HTML5에서 이미지 리소스를 지정 할 때 유연성을 높이기 위해 도입댔다.
브라우저는 <source>요소에서 media속성값의 크기에 맞춰 이미지를 출력한다.

브라우저 창 (뷰포트)이 최소 650 픽셀 인 경우 하나의 사진을 표시하고 그렇지 않은 경우 465 픽셀보다 큰 다른 이미지를 표시한다.

결과 화면

<picture> 요소에 마지막 요소는 항상 <img> 요소로 지정해야 되는 이유는 HTML5를 지원하지 않는 브라우저에서 <img> 요소는 지원되지만 <picture> 요소는 지원하지 않기 때문에 <source>태그와 매치를 시킬 수 없다.
