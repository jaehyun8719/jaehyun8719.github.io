HTML5 레이아웃(Layout)

HTML 레이아웃 요소

웹 사이트는 잡지나 신문과 같이 여러 열로 내용을 표시한다.
HTML은 웹페이지의 다른 부분을 정의하는 몇 가지 의미 요소를 제공한다.

  • <header>-문서 또는 섹션의 헤더를 정의한다.
  • <nav>-탐색 링크를위한 컨테이너를 정의한다.
  • <section>-문서에서 섹션을 정의한다.
  • <article>-독립적 인 내용을 정의한다.
  • <aside>-사이드 바와 같이 내용을 정의한다.
  • <footer>-문서 또는 섹션의 바닥 글을 의한다.
  • <details>-추가 세부 사항을 정의한다.
  • <summary>-<details>요소의 제목을 정의한다.

HTML 레이아웃 기법

여러 열 레이아웃을 만드는 5가지 방법

  • HTML table(권장하지 않음)
  • CSS flaot 속성
  • CSS flexbox
  • CSS framework
  • CSS grid

HTML table(권장하지 않음)

<table>요소는 레이아웃 도구로 설계되지 않았다. <table>요소의 목적은 표 형식의 데이터를 표시 하는 것이다. 레이아웃에 <table>을 사용 하면 안된다.

CSS framework

레이아웃을 빠르게 만들려면 W3.CSS 또는 Bootstrap 과 같은 프레임 워크를 사용할 수 있다.

CSS floats

CSS float 속성을 사용하여 전체 웹 레이아웃을 만드는 것이 일반적이다. 단점으로는 float 요소가 문서 흐름과 연결되어있어 유연성이 떨어질 수 있다.

CSS flexbox

flexbox는 CSS3의 새로운 레이아웃 모드이다.
flexbox를 사용하면 페이지 레이아웃이 다른 화면 크기와 다른 디스플레이 장치를 수용해야 할 때 예측 가능하게 작동 한다. 단점으로는 IE10 이하 버전에서는 작동하지 않는다.
