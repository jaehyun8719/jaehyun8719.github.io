HTML 문단

HTML <p>요소는 문단을 정의한다.

HTML요소는 공백을 추가하여 결과를 나타낼수 없다. 브라우저가 공백을 제거해서 화면에 표시 해주기 때문이다.

종료 태그를 사용

대부분의 브라우저에서는 종료 태그를 사용하지 않아도 정상적으로 화면에 출력이 된다.

종료 태그를 사용하지 않으면 생각지 못한 오류가 발생할 수 있다.

HTML 줄 바꿈

HTML <br>요소는 줄바꿈을 정의한다. 새 문단을 시작하지않고 줄 바꿈을 할때 사용하면 된다.


See the Pen
HTML5 05.HTML 줄바꿈 by Jaehyun (@jaehyun8719)

<br> 요소는 종료 태그가 없는 빈 태그 이다.

HTML <pre>요소

HTML <pre>요소는 내가 작성한 내용을 그대로 표현한다. 즉, <br>태그와 다르게 공백유지, 문단 유지를 시켜준다.
