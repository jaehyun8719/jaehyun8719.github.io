HTML5 요소(Elements)

HTML 요소(Elements)

HTML요소는 start tag(시작 태그)와 end tag(종료 태그)로 이루어진 모든 명령어들을 의미한다.
<p>My first paragraph.</p>

Start tag Element content End tag
<p> My first paragraph. </p>
<br>

내용이 없는 요소들은 End Tag(종료 태그)가 없다.

중첩된 HTML 요소

HTML에는 요소들이 중첩 될 수 있다. 모든 HTML 은 중첩된 HTML 요소로 구성되어있다.


See the Pen
HTML5 02. 중첩 된 HTML 요소 by Jaehyun (@jaehyun8719)
on CodePen.

이 예제에서는 4개의 요소가 포함되어 있다.

예제 설명

<html> 요소는 문서 전체를 나타내며, 시작 태그 <html>, 종료 태그 </html> 요소 안에 <body>가 있다.

<body> 요소는 문서 본문을 나타내며, 시작 태그 <body>, 종료 태그 </body> 요소 안에 2개의 다른 html 요소(<h1>, <p>) 가 있다.

<h1> 요소는 제목을 나타내며, 시작 태그 <h1>, 종료 태그 </h1>
요소 안에 내용은 My First Heading. 이다

<p> 요소는 단락을 나타내며, 시작 태그 <p>, 종료 태그 </p> 요소 안에 내용은 My first paragraph. 이다

종료 태그를 잊지 말자!

종료 태그를 안 써도 일부 HTML 요소가 정상적으로 표시 된다.


See the Pen
HTML 5 02. 종료 태그를 잊지 말자! by Jaehyun (@jaehyun8719)
on CodePen.

정상 작동하는 것을 확인 할 수 있다. 브라우저에서 종료 태그는 선택 사항으로 간주 되는데 종료 태그를 잊어 버릴 경우 생각지 못한 오류가 발생 한다.

비어있는 HTML 요소

내용이 없는 HTML 요소를 비어있는(Empty) 요소라고 한다.
<br>태그는 종료 태그가 없는 비어있는 요소 이다. (<br>태그는 줄 바꿈을 나타내는 태그 이다.)


See the Pen
HTML5 02.비어있는 HTML 요소 by Jaehyun (@jaehyun8719)
on CodePen.

비어있는 요소는 시작 태그에서 </br> 이와 같이 닫을 수 있다. HTML5에서 닫을 필요는 없지만 더 엄격하게 관리를 원한다면 닫는 것이 좋다.

HTML 태그는 대소문자를 구분하지 않는다.

HTML 태그는 <P>와 <p>와 같다. 하지만 W3C는 소문자를 권장한다.
